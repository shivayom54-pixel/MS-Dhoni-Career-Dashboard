🏏 Mahendra Singh Dhoni Career Statistics Dashboard Analysis
1. Project Title

Mahendra Singh Dhoni Career Statistics Dashboard

An interactive Power BI dashboard developed to analyze the international cricket career of Mahendra Singh Dhoni across ODI, Test, and T20 formats. The dashboard highlights batting performance, scoring trends, boundaries, centuries, and yearly achievements.

2. Short Description / Purpose

The Mahendra Singh Dhoni Career Statistics Dashboard provides a complete overview of Dhoni's batting career. It helps users analyze his runs, batting average, strike rate, highest score, boundaries, centuries, and yearly performance through interactive visualizations.

The dashboard is useful for cricket fans, sports analysts, and data visualization enthusiasts who want to explore Dhoni's cricket journey through data-driven insights.

3. Tech Stack
📊 Power BI Desktop – Dashboard design and visualization.
📂 Power Query – Data cleaning and transformation.
🧠 DAX (Data Analysis Expressions) – Calculated measures and KPIs.
📁 Excel/CSV Dataset – Source cricket statistics data.
🔗 Data Modeling – Relationship management and analysis.
4. Data Source
